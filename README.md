# WebCraft

WebCraft is a simple web server that serves static files from a directory. It is written in C++ and works off of OS librariess and aims to be cross platform.

Currently, it is a work in progress and is not yet ready for use.
Things that have been implemented:
	-  Debugging, Logging, and Error Handling frameworks
		- View webcraft/util/debug.h for more information
	-  Unit testing framework
		- View webcraft/util/unit_test.h for more information
